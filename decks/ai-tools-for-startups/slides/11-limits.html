So here's why it does weird things

!

Confident hallucinations

It generates what's plausible, not what's true. A fake citation looks statistically like a real one — so it writes one with total confidence.

#

Bad at exact math & counting

It's pattern-matching language, not calculating. "How many r's in strawberry" is hard because it sees tokens, not letters. Give it a tool or calculator.

~

No memory of its own

Each request only "knows" what's in the prompt — the context window. Close the chat and it forgets. Memory features just re-feed past text back in.

@

Frozen knowledge + bias

It only knows its training data, up to a cutoff date, and absorbs that data's biases. For current or proprietary info, it needs search or your documents fed in.
