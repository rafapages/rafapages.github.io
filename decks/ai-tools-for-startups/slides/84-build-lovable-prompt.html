Lovable New project

Let's create a webapp for my new company. It's going to be a project management tool for architects who do home renovations for clients. The tool needs to have:

  • A Kanban board with columns of cards. Cards need to be editable, and have a creation date and due date. Columns need to be editable too. We should be able to enter text in the cards, and also comments.
  • A timeline view where all the tasks in the Kanban board are represented.
  • A wiki view where the architect can drop information on providers.
  • A moodboard view the architect can compile with ideas for their clients.

We will add login information later, but let's start with this general view. Include some sample data so we can get an idea of how it works all together. Make sure its UI is nice and simple, with minimal design elements, as architects typically like.
